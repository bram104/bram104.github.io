Project Hosting

Bij project hosting is het de bedoeling om een cluster te maken om de projecten van de 2de jaars Applicatie Development studenten op te kunnen hosten. Wij als groepje hebben beslist om hiervoor rancher te gebruiken als oplossing. Rancher is een product dat kubernetes makkelijk kan managen. Hierdoor is het dus voor ons makkelijk om nieuwe kubernetes components makkelijk op te kunnen spinnen.

Nextcloud

Dit is een project dat ik heb verwezelijkt tijdens het vak 'Linux Server'. Tijdens dit project heb ik een oude computer thuis omgezet naar een linux server met een werkende nextcloud op. Met nextcloud heb je de mogelijkheid om je eigen cloud storage service te maken op je eigen server omdat alles open source is. nextcloud heeft veel geavanceerde functie's ook dus het dient niet enkel om je bestanden in kwijt te kunnen.

IoT Mouse Trap

Dit is een project waar ik op dit moment nog aan bezig ben, dit is een project van het vak 'IoT Essentials'. De bedoeling is om een slimme muizenval te maken met een Ultrasone sensor en een stappenmotor die de deur langs de voorkant van de doos laat dichtvallen als de ultrasone sensor iets detecteert.
